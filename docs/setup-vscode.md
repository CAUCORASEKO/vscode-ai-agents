# Using these agents in VS Code

This setup assumes:
- VS Code
- Open Codex Sidebar
- ChatGPT (Plus or Team)

## Steps

1. Open VS Code
2. Open Open Codex Sidebar
3. Create a new chat
4. Name it after the agent (e.g. "React Performance Agent")
5. Paste the agent definition as the first message
6. Never reuse this chat for other purposes
